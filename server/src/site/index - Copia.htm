OrientDB Server

The OrientDB Server accepts both binary and RESTful/HTTP requests. In this page you can execute the basic commands to get acquainted with. The database 'demo' is persistent while 'temp' resides only in memory.

Direct URL: http://<server>:<port>/<command>/<database>/<operation>
