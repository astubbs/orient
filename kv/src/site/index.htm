Welcome in the Orient KeyValue Server homepage.
To start using this Orient KeyValue you can store your information just using the HTTP protocol. Example:
  1. HTTP POST to http://localhost:8000/petshop/demo/1 passing the value "Test1"
  2. HTTP GET to http://localhost:8000/petshop/demo/1: you'll receive back the value inserted, namely "Test1"

Syntax:
http://<server>:<port>/<bucket>/<key>

HTTP Method Purpose Note Try it
POST Insert the content in the bucket with the key specified If the key already exists, an error will be returned.

GET Retrieve the content of the key in the bucket specified If the key doesn't exists an error will be returned.
